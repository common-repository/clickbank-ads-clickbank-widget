== Description ==

Affiliate marketing is a type of performance-based marketing in which a business rewards one or more affiliates for each visitor or customer brought by the affiliate's own marketing efforts.
This Affiliate plugin creates a banner to display Vacation Rentals graphic affiliate ads on your Wordpress blog.
= You will receive 7% of the base rent price. =
If a visitor goes to our platform and books a vacation, pays, and completes his stay, then at the time of departure, the affiliate reward will become available for payout to you.

== Installation ==

Installation from within WordPress

1. Visit Plugins > Add New.
2. Search for "Affiliate Ads for cbAds.com".
3. Install and activate the "Affiliate Ads for cbAds.com" plugin.

Manual installation

1. Download the ZIPped plugin files.
2. Go to your WP admin page to `Plugins` => `Add New`. Click to `Upload` link, select ZIPped files and install plugin.
3. Click to `Activate plugin` link.

For post area:

1. Go to the `Settings -> Affiliate Ads for cbAds.com` SubPanel and configure plugin.
2. Fill out the fields, or leave them as the defaults
3. Click `Save` and view your site and ads

For wigget area:

1. Go to the `Appearance -> Widgets` SubPanel to add the "Affiliate Ads for cbAds.com" to your sidebar and configure it.
2. Fill out the fields, or leave them as the defaults
3. Click `Save` and view your site and ads

The ad block can be inserted using the shortcode API: [cbads]
Place [cbads] in any post, page, or widget to display the ad block.

Tested with PHP 8.2
